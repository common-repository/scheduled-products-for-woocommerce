Very simple plugin for scheduling publishing and unpublishing of WooCommerce products.

== Description ==

Scheduled Products for WooCommerce is a very simple plugin for scheduling publishing and unpublishing of WooCommerce products. You can schedule when product gets published and/or unpublished. There is no countdown timers, notifications or anything fancy. Just very simple publish or unpublish at certain time.

= Features =

* Schedule publishing and unpublishing of a product
* Works for simple products and variations
* Works with selected WordPress timezone

== Installation ==

Scheduled Products for WooCommerce is installed just like any other WordPress plugin.

1. Download the plugin zip file
2. Go to Plugins in the WordPress admin panel
3. Click Add new and Upload plugin
4. Choose the downloaded zip file and upload it
5. Activate the plugin

Once the plugin is activated, you should see "Schedule" in the product tabs. Please see the screenshots below.
